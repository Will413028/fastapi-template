# fastapi-template

## Quick Start

### 1. Install Pre-commit
```bash
pre-commit install
```

### 2. Create Virtual Environment

```bash
poetry shell
```

### 3. Install Dependencies

```bash
poetry install
```

### 4. Run Server

```bash
make run
```

## Code lint

```bash
make lint
```

## Run Tests

```bash
make test
```

## Export Requirements
```bash
make lock
```

## Deploy to remote server
```bash
make deploy
```

## Database Operations

### Auto-generate Migration

```bash
alembic revision --autogenerate -m "migration message"
```

### Database Migration

```bash
alembic upgrade head
```

## Docker

### Build Image

```bash
sudo docker build -t <image_name>:<tag> .
```

### Run Docker Compose

```bash
sudo docker-compose up -d --build
```

### Add Dependency
```bash
uv add <library_name>
```

### Add Dependency to dev
```bash
uv add --dev <library_name>
```
